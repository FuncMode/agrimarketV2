Privacy Policy

Patakaran sa Privacy

Pinahahalagahan ng AgriMarket ang data privacy at security ng mga gumagamit nito.

Mga Impormasyong Kinokolekta

Personal Profile Data

Pangalan, contact number, address, at iba pang profile information na ibinigay mo during registration.

Uploaded Verification Documents

Government-issued ID at selfie para sa verification purposes.

Order at Message Logs

Transaction history, messages, at order details (admin-accessible lamang para sa dispute resolution).

Data Protection Measures

Encrypted Storage

Ang IDs at images ay stored with encryption

Role-Based Access

Limited access based on user roles

Admin-Only Sensitive Records

Verification documents at sensitive data ay accessible ng admin lamang

Paggamit ng Data

Ang personal data ay ginagamit lamang para sa:

  • System operation at user authentication
  • Verification at security purposes
  • Academic evaluation ng project

Third-Party Sharing

Ang personal data ay HINDI ibinabahagi sa third parties. Ang lahat ng impormasyon ay ginagamit lamang within the system at para sa academic purposes.

Karapatan ng User

Bilang user, may karapatan ka sa:

Access sa iyong personal information
Correction ng mali o outdated na data
Request for account deletion

Academic Context

Dahil ang AgriMarket ay isang academic prototype, ang lahat ng collected data ay gagamitin lamang para sa system development, testing, at academic evaluation. Walang commercial use ng data.
